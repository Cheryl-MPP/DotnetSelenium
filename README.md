# DotnetSelenium

# Selenium C# .NET UI Automation Demo

This project demonstrates automated UI testing using **Selenium WebDriver with C# (.NET) and NUnit**.  
It includes interaction with common web controls such as dropdown lists and multi-select elements using a local HTML test page.

The purpose of this repository is to showcase automation skills and provide a simple reference implementation for Selenium beginners.

---

## Features

- Automated browser interaction with Chrome
- Dropdown selection using Selenium `SelectElement`
- Multi-select interaction
- Explicit waits with WebDriverWait
- Local HTML test environment (no external website required)
- NUnit test execution
- Console output validation

---

## Technologies Used

- .NET 8 / .NET 10
- C#
- Selenium WebDriver
- Selenium ChromeDriver
- NUnit
- Visual Studio / VS Code

# **Test Scenario**

-The automation performs the following steps:

-Opens a local HTML page

-Enters text into a textbox

-Selects an option from a dropdown

-Selects multiple values from a multi-select control

-Clicks the submit button

-Validates the result displayed on the page

# **Learning Goals**

-Basic Selenium automation in C#

-Working with dropdown and multi-select controls

-Synchronization using explicit waits
